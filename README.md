# Calculator App

A simple, elegant web-based calculator application built with HTML, CSS, and JavaScript.

## Features

- ✨ Clean and modern user interface
- 🧮 Basic arithmetic operations (+, −, ×, ÷)
- 📊 Modulo (%) operation
- 🎯 Decimal point support
- ⌨️ Full keyboard support
- 🗑️ Delete and Clear functions
- 📱 Responsive design (mobile-friendly)
- 🎨 Beautiful gradient background and smooth animations

## How to Use

1. **Open the app**: Open `index.html` in your web browser
2. **Enter numbers**: Click number buttons or use your keyboard
3. **Choose operation**: Click an operator button or use keyboard symbols
4. **Calculate result**: Click `=` or press Enter
5. **Clear**: Click `C` or press Escape to clear all
6. **Delete**: Click `DEL` or press Backspace to remove last digit

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0-9` | Input numbers |
| `.` | Decimal point |
| `+` `-` `*` `/` | Operators |
| `Enter` or `=` | Calculate |
| `Escape` | Clear all |
| `Backspace` | Delete last digit |
